Tool Name : UpdateWebConfig
Tool Purpose : This tool will update the webconfig file from pointing to 2.7 Env (http://10.251.60.19:8080) to 2.9 Server(https://simpos-txn-cert.starbucks.net:443)

How to use this tool: 
1. Copy 2 files UpdateWebConfig.ps1 and RunUpdateWebConfig.bat files into C:\temp on your target machine.
2. Open command prompt
3. Run RunUpdateWebConfig.bat file and your webconfig should be updated.
